# Database Studies Repository

This repository contains **database scripts developed for study and practice purposes**.
The goal of this repository is to organize and store different **database models, SQL scripts, and experiments** created while learning concepts related to **database design, relational modeling, and SQL programming**.
Each project represents a small scenario designed to practice specific database concepts.

# Purpose of This Repository

This repository was created to:

* Practice **SQL syntax**
* Develop **relational database models**
* Apply **primary and foreign keys**
* Understand **relationships between entities**
* Experiment with **database structure modifications**
* Improve skills in **data organization and integrity**

# What You Will Find Here

This repository may include:

* Database creation scripts
* Table creation (`CREATE TABLE`)
* Relationship definitions
* Structural changes using `ALTER TABLE`
* Constraints such as:

  * `PRIMARY KEY`
  * `FOREIGN KEY`
  * `UNIQUE`
  * `DEFAULT`
* Example database models for study

Each folder or file may represent a **different database scenario** used for learning purposes.

# Technologies Used

* SQL
* Relational Database Modeling
* Database Design Concepts

# Learning Focus

The main focus of this repository is practicing:

* Database creation
* Table relationships
* Data integrity rules
* Schema evolution
* Query experimentation

Each folder contains **a specific database study project**.
