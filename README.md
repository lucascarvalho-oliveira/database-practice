# Repositório de Estudos de Banco de Dados

Este repositório contém **scripts de bancos de dados desenvolvidos para fins de estudo e prática**.
O objetivo deste repositório é organizar e armazenar diferentes **modelos de banco de dados, scripts SQL e experimentos** criados durante o aprendizado de conceitos relacionados a **modelagem de dados, bancos de dados relacionais e SQL**.
Cada projeto representa um pequeno cenário criado para praticar conceitos específicos de banco de dados.

# Objetivo do Repositório

Este repositório foi criado para:

* Praticar **sintaxe SQL**
* Desenvolver **modelos de banco de dados relacionais**
* Aplicar **chaves primárias e estrangeiras**
* Compreender **relacionamentos entre entidades**
* Experimentar **modificações na estrutura do banco**
* Aprimorar habilidades em **organização e integridade de dados**

# O Que Você Vai Encontrar Aqui

Este repositório pode incluir:

* Scripts de criação de banco de dados
* Criação de tabelas (`CREATE TABLE`)
* Definição de relacionamentos
* Alterações estruturais usando `ALTER TABLE`
* Restrições de integridade como:

  * `PRIMARY KEY`
  * `FOREIGN KEY`
  * `UNIQUE`
  * `DEFAULT`
* Modelos de banco de dados usados para prática

Cada pasta ou arquivo pode representar **um cenário diferente de estudo**.

# Tecnologias Utilizadas

* SQL
* Modelagem de Banco de Dados Relacional
* Conceitos de Design de Banco de Dados

# Foco de Aprendizado

O foco principal deste repositório é praticar:

* Criação de bancos de dados
* Relacionamento entre tabelas
* Regras de integridade de dados
* Evolução da estrutura do banco
* Experimentação com consultas SQL

# Estrutura do Repositório

Cada pasta contém **um projeto de banco de dados voltado para estudo**.

Eles não foram desenvolvidos para uso em produção.
